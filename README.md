# Walls Model: Bidirectional Template Following

Простой веб-сервис на **HTML/CSS/JS + Python (Flask + OpenCV)**, который реализует идею:

- пользователь выбирает цвет целевой стены,
- выделяет эталонный фрагмент (ROI),
- система двигает этот шаблон маленькими шагами в обе стороны,
- на каждом шаге проверяет цвет, форму, края и толщину,
- останавливается при потере уверенного совпадения.

## Запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

После этого откройте: `http://127.0.0.1:5000`

## Как пользоваться

1. Загрузить изображение.
2. Нажать **"1) Пикнуть цвет"** и кликнуть цвет нужной стены.
3. Выделить мышью небольшой ROI на линии стены.
4. Нажать **"2) Запустить"**.
5. Смотреть результат: зелёные окна позиций шаблона + жёлтая осевая траектория.

## Алгоритм (базовая версия)

- Из ROI строится маска по выбранному цвету (с допуском).
- По маске оценивается главная ось (PCA) — базовое направление движения.
- Для каждой итерации в обе стороны:
  - считаем ожидаемую новую позицию,
  - выполняем локальный поиск вокруг неё,
  - считаем интегральный score (цвет + края + толщина),
  - принимаем лучшую локальную посадку,
  - при серии плохих совпадений останавливаемся.

> В данной версии траектория — линейная (PCA-guided). Для дуг можно расширить модель, добавив оценку кривизны и локальный поворот шаблона.
